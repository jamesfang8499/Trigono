# 北京四中高中教学讲义  三角（全一册）
本项目对1995年4月出版发行的《北京四中高中教学讲义——三角（全一册）》进行了重排。这套讲义包括六册：高中代数第一、二、三册，三角、立体几何、解析几何各一册。

原书的末尾附有习题答案，此次重排将这块的内容单独出来，形成一个新的项目。

书中的矢量图片使用Tikz和Tkz-Euclide制作。

注意：本项目的内容勿用于商业目的。
